# ESP32 library to control Omega Split-Flap Display

## Test board connections

GPIO 19 -> ULN2803 -> Start
GPIO 18 -> ULN2803 -> ADL 
GPIO 5  -> ULN2803 -> ADC hours
GPIO 17 -> ULN2803 -> ADC minutes
 
GPIO 32 -> DATA 0
GPIO 33 -> DATA 1
GPIO 25 -> DATA 2
GPIO 26 -> DATA 3
GPIO 27 -> DATA 4
GPIO 14 -> DATA 5
